The Sorting Visualizer

Sorting Visualization is a pictorial and graphical representation of different sorting techniques. Through this we try to put forward an imagination of learner while studying sorting which can help one to understand visually the sorting techniques. New learners can refer to the learning section to learn basics of sorting.

The user can either view each step one at a time or can choose the option to the sorting animation. To make the user understand better, different color codes are used for different array elements. Hope that by using this tool the user can uderstand more clearly, the basic sorting algorithms that are included.

This project would not have been possible, without the guidance and useful advice of our mentor Mr. Pawan Pant.
